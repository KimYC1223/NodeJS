# Pizza Maker Web Server

## INIT

```npm install```

## START

```node server.js```

---

## Enter Web Page

- ```http://localhost:15000/``` : 인덱스 페이지
- ```http://localhost:15000/UserOrderPage``` : 버튼 조작 할수 있는 페이지
- ```http://localhost:15000/UserIndexPage``` : 카카오 지도로 기계 선택 할 수 있는 페이지

자세한 내용은 ```routes.js```확인 바람
